🛒 Product List With Cart

A simple and responsive Product List with Add to Cart web application built using HTML, CSS, and JavaScript.

🚀 Live Features

🔹📦 Display product list dynamically

🔹🛍️ Add to Cart functionality

🔹➕ Increase / ➖ Decrease product quantity

🔹🗑️ Remove item from cart

🔹💰 Auto total price calculation

🔹📱 Fully responsive design

🛠️ Technologies Used

🔹HTML5

🔹CSS3

🔹JavaScript (Vanilla JS)

📂 Project Structure
product-list-with-cart/
│
├── index.html
├── style.css
├── script.js
└── README.md

⚙️ How It Works

1. Products are displayed on the homepage.

2. When user clicks Add to Cart, the item is added to the cart section.

3. User can increase or decrease quantity.

4. Total price updates automatically using JavaScript.
